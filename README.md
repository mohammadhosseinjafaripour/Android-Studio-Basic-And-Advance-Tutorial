# Android Studio Basic and Advance Tutorials

Collection of Android Studio sample projects. Each folder is an independent
Android Studio project that can be opened and run on its own.

## Structure
- One repo, many standalone projects.
- Open any folder directly in Android Studio.

## Open a project
1. Open Android Studio.
2. File -> Open.
3. Select the project folder (for example `BottomDialog`).

## Add new samples
- Create a new project in Android Studio.
- Move the project folder into this repo root.
- Keep each project in its own folder.
